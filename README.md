# Mission Board Web Application

## Overview
This application serves as an executive dashboard for task management and progress tracking toward operational goals.

## Features
- Kanban-style task management for two categories: 'Jose' tasks and 'Orbit' tasks.
- Progress tracking with a visual progress bar based on task completion.
- Real-time updates using localStorage for data persistence.
- Responsive design for desktop and mobile viewing.
- Integration hooks for Telegram bot notifications.

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Deployment
Deploy the application to Vercel using the Vercel CLI or through the Vercel website by linking the GitHub repository.

## Usage
Navigate to the appropriate URL as provided by the Vercel deployment to access the application. Adjust tasks and view progress as needed.
